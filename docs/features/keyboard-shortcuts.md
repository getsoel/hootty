# Keyboard Shortcuts

All keyboard shortcuts available in Hootty.

## Shortcuts

| Shortcut | Action | Menu |
|----------|--------|------|
| `Cmd+T` | New Tab | Shell |
| `Cmd+D` | Split Right | Shell |
| `Cmd+Shift+D` | Split Down | Shell |
| `Cmd+Option+D` | Split Left | Shell |
| `Cmd+Option+Shift+D` | Split Up | Shell |
| `Cmd+Shift+S` | Toggle Sidebar | View |

## Terminal keys

All other key presses are forwarded directly to the terminal. This includes:

- `Escape` — always sent to the terminal (never closes the window).
- Standard shortcuts like `Cmd+C` / `Cmd+V` for copy/paste.
- Modifier combinations (`Ctrl`, `Option`, `Shift`, `Cmd`) are passed through to the shell and applications.
- Input Method Editor (IME) composing sequences for CJK and accented character input.
